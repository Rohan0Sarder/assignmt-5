Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer : Using innerHTML the user can see the content inside the HTML , Using innerText users can only watch texts and Using textContent user can see only the text content

Question-3: What is event delegation in the DOM?

Answer : যখন আমরা কোন বাটন বা এমন যাতের কোন কিছুকে ক্লিক করলে আউটপুট হিসেবে কিছু কাজ হবে তখন এই event delegration টা ব্যাবহার করি

Question-4: What is event bubbling in the DOM?

Answer : Event Bubbling , Event Delegation ব্যবহারে সাহায্য করে, এর মাধ্যমে আমরা parent element-এ একটি addEventListener করেই তার সব child elements-এর ইভেন্টকে কন্ট্রোল করতে পারি
